# Pong

A Java implementation of Pong.

## Installation

1. Ensure the [Java Development Kit](https://www.oracle.com/java/technologies/downloads/) is installed on your system.

2. Clone the repository to your desired location.

3. Navigate to the `src` directory.

4. Compile the program.
   ```Java
   javac Main.java
   ```

5. Run the program.
   ```Java
   java Main
   ```
